# Расчет квадратных уравнений
Веб-приложение для вычисления корней квадратного уравнения вида ax^2 + bx + c = 0

Пример использования: http://localhost:5000/api/v1/quadratic-equation?a=2&b=7&c=1

## Технологии
* Spring Boot 2.7.5
  * Spring Web MVC;
  * Spring Data JPA;
* СУБД: PostgreSQL, H2;
* Maven 3
* JDK 17
